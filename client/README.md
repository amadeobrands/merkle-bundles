astex
=====

Merkle AST Bundles. 

Includes:
 - client
 - server

Uses Karma/Mocha for unit tests of server and client, and E2E testing with Nightwatch.

Webpack is used for bundling. Express for server.

```
merkle-ast-serve -d "." -a 127.0.0.1 -p 3000
```
